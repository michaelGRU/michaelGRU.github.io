The Java program compares different text documents and estimates the probability that the documents are written by the same author. The script calculates the similarity between the vocabulary used based on unique words to word count ratio and common words the two input documents share. The algorithm is optimized for large data sets.(Source Code: text_comparison.java)

Note: To run the file, you need to have a Java compiler on your computer. You can download it here: Java SE Downloads
You can then execute the program directly from the command line.

Enter the file names. Here, I downloaded two text files from Project Gutenberg. Make sure you save the java file, and the two text documents you want to compare in the same folder.
